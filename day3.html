HTML Lists

HTML provides several types of lists:

  • Ordered Lists
  • Unordered Lists
  • Description Lists

Example of an Ordered List

  1. Wake up
  2. Take a shower
  3. Have breakfast
  4. Brush teeth
  5. Go to class

Example of an Unordered List

  • Apples
  • Bananas
  • Cherries

Example of a Description List

Images

Images are added using the <img> tag. Example:

Attributes:

  • src: Specifies the path to the image
  • alt: Provides alternative text for the image
  • width and height: Define the dimensions of the image

References

  • MDN Web Docs - <img> element
  • W3Schools - HTML Images

Copyright Concerns

When using images from the web, it's important to consider copyright issues. Always check the image's license and give proper attribution if required.

Audio

Audio files can be embedded using the <audio> tag. Example:

From https://commons.wikimedia.org/wiki/File:1-06_Lobachevsky.mp3

Attributes

  • controls: Adds audio controls like play, pause, and volume
  • autoplay: Starts playing the audio as soon as it is ready
  • loop: Repeats the audio indefinitely

References

  • MDN Web Docs - <audio> element
  • W3Schools - HTML Audio

Tables

Tables are created using the <table> tag along with related tags like <tr>, <th>, and <td>. Example:

Monday Math Class 9:00 AM - 10:30 AM
Tuesday Science Class 10:00 AM - 11:30 AM
Wednesday History Class 1:00 PM - 2:30 PM

References

  • MDN Web Docs - <table> element

Styling Tables

Tables can be styled using CSS to improve their appearance and readability.

In later lessons will learn about various CSS properties that can be applied to tables, such as border, padding, and background-color.
