HTML Lists

HTML provides several types of lists:

  • Ordered Lists
  • Unordered Lists
  • Description Lists

Example of an Ordered List

  1. Wake up
  2. Take a shower
  3. Have breakfast
  4. Brush teeth
  5. Go to class

Example of an Unordered List

  • Apples
  • Bananas
  • Cherries
